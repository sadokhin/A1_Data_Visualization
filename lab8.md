| Название столбца | Значение |
| -----------------|:--------:|
| Order ID | Идентификатор заказа |

| Quantity Ordered |	Количество |
| Price Each |	Цена |
| Order Date | Дата заказа |
| Purchase Address | Адрес покупателя |
